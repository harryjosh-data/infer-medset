Med Datasets CLI

A command-line interface for accessing medical datasets.

Installation

For macOS/Linux users:

$ curl -s https://med-datasets-cli.netlify.app/installers/bootstrap-unix.sh | bash

For Windows users:

Download and run the installer:

Download Windows Installer

Usage

After installation, simply type the following command in your terminal:

$ infer-medset

Features

  • 🔍 Explore Datasets - Browse available medical datasets
  • 📊 Access Datasets - Access and download datasets for your research
  • 💻 Easy to use command-line interface with intuitive navigation

Requirements

  • Node.js version 14 or higher
